Объявление функции

1.1 - объявление функции с помощью инструкции

    function f (param) {
        console.log(param)
    }

1.2 - объявление функции с помощью выражения

    var a = function f (param) {
        console.log(param)
    }

1.3 - объявление функции с помощью конструктора

    
        var a = new Function ('param', 'console.log(param)');

1.4 - Тесты

    function f () {
        console.log(f); // 1
    }

    var a = function b () {
        console.log(b)  // 2
    }

    console.log(f) // 3
    console.log(b) // 4

Для функции, объявленной с помощью инструкции ее имя доступно во всей области видимости
